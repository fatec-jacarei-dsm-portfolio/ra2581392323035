## 🚀 Projetos

### 1. Sistema Kanban de Produtividade
Quadro Kanban com drag-and-drop para organização visual de tarefas e fluxos de trabalho.
→ [`/projetos/P3`](./projetos/P3) · [Repositório externo](https://github.com/Lucas-assis1/P3-LDW)

```bash
cd projetos/P3
npm install
npm run dev
```

### 2. App de Rotina de Saúde
Aplicativo mobile para acompanhamento diário de macronutrientes (proteínas, carboidratos, gorduras).
→ [`/projetos/P3-PDM-main`](./projetos/P3-PDM-main) · [Repositório externo](https://github.com/Lucas-assis1/P3-PDM)

```bash
cd projetos/P3-PDM-main
npm install
npx expo start
```

### 3. Pipeline de Conciliação Financeira
Compara extrato bancário com lançamentos do ERP e gera relatório Excel de divergências.
→ [`/projetos/conciliacao-financeira`](./projetos/conciliacao-financeira)

```bash
cd projetos/conciliacao-financeira
pip install -r requirements.txt
python src/conciliacao.py
```

### 4. Modelagem de Forecast de Receita
Modelo SARIMA que projeta receita dos próximos 12 meses com IC 95%.
→ [`/projetos/forecast-receita`](./projetos/forecast-receita)

```bash
cd projetos/forecast-receita
pip install -r requirements.txt
python src/forecast.py
```

### 5. Dashboard de Performance Financeira
Queries SQL prontas para conectar ao Power BI: DRE, inadimplência e fluxo de caixa.
→ [`/projetos/dashboard-performance`](./projetos/dashboard-performance)

### 6. Snake Game
Recriação do clássico jogo da cobrinha com JavaScript puro e HTML Canvas.
→ [`/projetos/snake-game`](./projetos/snake-game)

### 7. Localizador de Cães
Plataforma para localização e adoção de cães perdidos, com geolocalização e cadastros colaborativos.
→ [Documentação](https://github.com/candago-5/documentacao)

### 8. Assistente de WhatsApp — Procon Jacareí
Bot conversacional para atendimento automatizado ao consumidor no Procon de Jacareí.
→ [Documentação](https://github.com/candago-6/documentacao)
