# Gerenciador de Tarefas por Categoria - React

Este é um aplicativo React simples para gerenciar tarefas organizadas por categorias. Ele inclui funcionalidades como adicionar, editar, filtrar e excluir tarefas, além de suporte a modo claro e escuro, com persistência no `localStorage`.

---

## Funcionalidades

- Criar, editar e excluir categorias de tarefas
- Adicionar, editar, concluir e excluir tarefas dentro das categorias
- Filtrar tarefas (todas, concluídas, pendentes)
- Reordenar categorias (mover para cima/baixo)
- Modo escuro/Claro com botão toggle no canto superior direito
- Persistência dos dados no `localStorage` para manter estado entre sessões

---

## Tecnologias Utilizadas

- React (com hooks: useState, useEffect)
- JavaScript (ES6+)
- CSS inline simples para estilização
- LocalStorage para persistência local
